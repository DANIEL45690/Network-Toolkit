## ✨ Возможности

| Категория | Команды |
|-----------|---------|
| 🌐 **Базовые сетевые** | `ping`, `tracert`, `nslookup`, `arp`, `route` |
| 📡 **IP конфигурация** | `ipconfig`, `ipconfig /all`, `/flushdns`, `/release`, `/renew` |
| 📊 **Сетевая статистика** | `netstat`, `netstat -a`, `-b`, `-n`, `-o` |
| 🖥️ **Системная информация** | `systeminfo`, `tasklist`, `driverquery`, `getmac`, `hostname`, `whoami` |
| 🔌 **Windows сети** | `netsh wlan`, `netsh firewall`, `net share`, `net user`, `net use` |
| 🚀 **Продвинутые инструменты** | `publicip`, `localips`, `portscan`, `asyncscan`, `dns`, `reversedns` |
| 🔗 **Современные протоколы** | `websocket`, `zeroconf` |
| 📜 **Утилиты** | `history`, `export`, `clear` |
